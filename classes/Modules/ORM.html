    Instance Method Details

    - (void) get(which)

    Note: Only properties added via the ORM can be matched

    A shorthand for retrieving an object via its ID, symlink (organic ID) or property criteria.

    Parameters:

    • which (String) — ID - the default ID created by the ORM to identify unique reccords

    Options Hash: (which):

    • ID (String) — - the organic ID given by the application / user / developer
    • An (Object) — object containing matchers to retrieve the object (passed to the #getAdv fnction)

    - (Object) getAdv(what)

    Note: Any first level item may refer to an object containing a modifier ($gt, $lt, $gte, $lte, $contains)

    Retrieves an object via property criteria

    Examples:

    How to find an object 

    ORM.getAdv Username: "Xulescu", Password: sha256("password", salt)

    How to find an object using modifiers 

    ORM.getAdv Username: "Xulescu", AuthLevel: { $gte: 5, $lte: 8 }

    Parameters:

    • what (Object) — The object containing all of the matchers

    Returns:

    • (Object) — The reccord requested or an array of reccords matching the given criteria.

    - (void) reuse(which, args)

    Note: The match argument is the same as in the #get function

    A shorthand for retrieving reccords. The requested reccord is firstly searched and, if found, retrieved. If the reccord is not found, it is created, taking the second parameter as arguments for the creation function.

    Parameters:

    • which (String) — The reccord to retrieve
    • args (Object) — The arguments to pass to the construction #create function.
