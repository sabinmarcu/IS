Class: Modules.Overload

Defined in: src/Modules/Overload.coffee

Overview

An overload engine, capable of replacing a function with a handler that decides on which function to use based on the arguments (and sets given)

Instance Method Summary

  • # (void) verifyAll(args...) Bound This function simply acts as a router, routing the handlers accordingly.
  • # (void) verify(what, how) Bound This function does the checks for any given set parameter

Constructor Details

# (void) constructor(sets, parent)

Constructing an overload manager, working with sets

// Will yield : One Argument / Two Arguments / Two Arguments and Object depending on the selection. ["something"] goes on the first, ["two", "args"] goes on the second, and [{something: "good"}, "and cool"] goes to the third set;

Examples:

Example of a set 

[
  one:
    if: args: 1
    then: -> "One argument"
  two:
    if: args: 2
    then: -> "Two arguments"
  two_and_object:
    if: 
      args: 2
      arg1: (arg) -> typeof arg is "object"
    then: -> "Two arguments and Object"
]

Parameters:

  • sets ( Array ) — The sets to be processed

# (void) verify(what, how) Bound

This function does the checks for any given set parameter

Parameters:

  • what ( String ) — The name of the argument (can be predefined : args, or it can be defined as : arg# where # is the number of the argument in order of appearance)
  • how ( Function ) — The handler function (or argumet in the case of predefined verifiers)
