Class: Modules.Observer

Defined in: src/Modules/Observer.coffee

Overview

The Observer class (or EventHandler) is a module that bestows the capability to triggevents, and subscribe to them to any given object. This practice is enspecially useful in JavaScript / CoffeeScript because if its event-driven nature.

Version:

  • v0.1

Instance Method Details

# (void) delegateEvent(event, handler, object = window)

Delegate an event to a given handler belonging to the given object.
If no object is supplied, then the window object is used.

Parameters:

  • event ( String ) — The event to which the handler will be delegated (or subscribed)
  • handler ( Function ) — The handler attached to the event.
  • object ( Object ) — The object to which the handler belongs to (delegate data from that event).

# (void) subscribe(event, handler)

Subscribe a handler belonging to the current object to a given event of itself.

Parameters:

  • event ( String ) — The event to which to delegate the handler.
  • handler ( Function ) — The handler to attach to the event.

# (void) publish(args...)

Publish (trigger) to the event, also pushing data at the same time.

Parameters:

  • event ( String ) — The event to publish to (trigger)
  • data ( Arguments... ) — Any data you wish to send through the channel to any handler attached to the event.
